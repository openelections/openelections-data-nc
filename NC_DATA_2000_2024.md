# North Carolina Precinct-Level Election Results (2000-2024)

## Data Source
North Carolina State Board of Elections (NCSBE)
- Original data downloaded from: https://www.ncsbe.gov/results-data/election-results
- Precinct-level results for general elections from 2000-2024

## Data Coverage
This contribution includes precinct-level results for the following North Carolina general elections:
- 2000 (November 7, 2000)
- 2002 (November 5, 2002)
- 2004 (November 2, 2004)
- 2006 (November 7, 2006)
- 2008 (November 4, 2008)
- 2010 (November 2, 2010)
- 2012 (November 6, 2012)
- 2014 (November 4, 2014)
- 2016 (November 8, 2016)
- 2018 (November 6, 2018)
- 2020 (November 3, 2020)
- 2022 (November 8, 2022)
- 2024 (November 5, 2024)

## Data Format
The data files contain the following fields:
- **County**: County name
- **Election Date**: Date of election
- **Precinct ID**: Numeric or alphanumeric precinct identifier
- **Precinct Name**: Full precinct name
- **Office**: Contest/office name
- **Candidate**: Candidate name
- **Party**: Party affiliation (DEM, REP, LIB, etc.)
- **Votes**: Vote count
- **Timestamp**: Data processing timestamp (where applicable)

## Data Processing
Some files have been processed for consistency:
1. **2008-2024**: CSV files cleaned for formatting issues (removed extra quotes, aligned columns)
2. **2000-2006**: Original format preserved from NCSBE with tab-delimited or aligned format
3. All files maintain original vote totals and candidate information without modification

### Processing Scripts
Python scripts used for data cleaning (for select years) are included:
- `clean_csv_2008.py`
- `clean_csv_2012.py`
- `clean_csv_2014.py`

These scripts primarily addressed:
- Inconsistent quote escaping
- Column alignment issues
- Character encoding problems

## File Naming Convention
Files follow NC State Board of Elections naming convention:
- `results_pct_YYYYMMDD.csv` - Original format
- `results_pct_YYYYMMDD_clean.csv` - Cleaned/processed format

## Data Completeness
- All 100 North Carolina counties included in each election
- Includes absentee/provisional votes (labeled as "absentee/provisional" precinct)
- Vote totals match official NCSBE certified results

## Notes
- Some precinct names/IDs changed between elections due to redistricting
- Early elections (2000-2006) use different format than later elections (2008-2024)
- All data represents official certified results from NCSBE
